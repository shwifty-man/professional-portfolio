## 🚀 Getting Started  

### Prerequisites  
- Node.js (>= 16 recommended)  
- npm or yarn  

### Installation  
```bash
# Clone the repo
git clone https://github.com/shwifty-man/professional-portfolio.git

# Navigate into the project
cd professional-portfolio

# Install dependencies
npm install  

# Run the development server
npm start  

```
